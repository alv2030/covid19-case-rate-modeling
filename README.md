# COVID-19 Case Rate Modeling

## Overview

This project analyzes county-level COVID-19 case rates across the United States using demographic, socioeconomic, educational, and population-based indicators.

The objective is to identify factors associated with COVID-19 infection rates and evaluate multiple statistical modeling approaches for count data.

## Dataset

The dataset contains observations from 1,275 U.S. counties and includes:
- COVID-19 case rates
- Population characteristics
- Education levels
- Economic indicators
- Demographic variables

## Methodology

- Exploratory Data Analysis
- Correlation Analysis
- Multiple Linear Regression
- Poisson Regression
- Quasi-Poisson Regression
- Negative Binomial Regression
